# The All Met

DC Sports & Tech. No Bullshit.

A bold, retro-styled publication covering Washington DC sports teams and technology culture.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4
- Markdown blog (gray-matter + remark)
- Google Fonts: Special Elite, Oswald, Courier Prime

## Project Structure

- `src/app/` — App Router pages
- `src/components/` — React components (layout, ui, home)
- `src/lib/` — Utilities (fonts, posts, constants)
- `content/blog/` — Markdown blog posts

## Deployment

Deploy on [Vercel](https://vercel.com) — push to main and it auto-deploys.
